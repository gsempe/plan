How to make an extension on all numeric types in Swift?

cf. [Numeric protocol](https://developer.apple.com/documentation/swift/swift_standard_library/numbers_and_basic_values/numeric_protocols)

```
extension Numeric {
    func addOne() -> Self {
        return self + 1
    }
}
1.addOne()
2.4.addOne()
```
